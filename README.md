# 🚦 Smart Traffic Management AI

An AI-powered Smart Traffic Management System that uses Computer Vision to optimize traffic flow through real-time vehicle detection, adaptive traffic signal control, emergency vehicle prioritization, red-light violation detection, and live analytics.

---

## 📖 Overview

Traditional traffic signals rely on fixed timers, which often lead to unnecessary congestion. This project uses Artificial Intelligence and Computer Vision to analyze traffic in real time and dynamically adjust signal timings based on vehicle density.

The system also detects emergency vehicles, prioritizes their movement, identifies red-light violations, and provides a live analytics dashboard for monitoring traffic conditions.

---

## ✨ Features

- 🚗 Real-time Vehicle Detection using YOLOv8
- 🎯 Vehicle Tracking with ByteTrack
- 🚦 Adaptive Traffic Signal Control
- 🚑 Emergency Vehicle Priority
- 🚨 Red-Light Violation Detection
- 📊 Live Analytics Dashboard
- 📈 Vehicle Distribution Charts
- 📉 Traffic Trend Analysis
- 🌐 Flask Web Interface

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/Smart-Traffic-Management-AI.git
```

Go to the project folder:

```bash
cd Smart-Traffic-Management-AI
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
